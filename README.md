# Nascor Collaboration UI

Plugin para WordPress que genera una interfaz interactiva de 4 columnas para escritorio y un formato 2x2 para dispositivos móviles. Incluye un panel de administración propio para gestionar fácilmente el diseño, los medios y los enlaces sociales.

## 🚀 Características Principales

* **Panel de Administración Custom:** Interfaz intuitiva ubicada en el menú de administrador de WordPress bajo "Nascor UI".
* **Personalización de Diseño:** Permite ajustar colores de fondo (superior e inferior) y la URL del logotipo central.
* **Integración Multimedia:** Soporta carga de imágenes y videos (MP4) con auto-reproducción inteligente basada en `IntersectionObserver`.
* **Carrusel Integrado:** Incluye un slider automático para la sección de "Arte".
* **Diseño 100% Responsivo:** Adaptable a pantallas grandes y dispositivos móviles.

## 🛠️ Instalación

1. Descarga el repositorio como un archivo `.zip`.
2. En tu panel de WordPress, ve a **Plugins > Añadir nuevo > Subir plugin**.
3. Sube el archivo `.zip` y haz clic en **Instalar ahora**.
4. Activa el plugin "Nascor Collaboration UI" desde el menú de Plugins.

## 💻 Uso

1. Ve a la nueva pestaña **Nascor UI** en el menú lateral de WordPress.
2. Configura tus colores, enlaces de redes sociales (LinkedIn, YouTube, TikTok, etc.) y URLs de imágenes/videos.
3. Guarda los cambios.
4. Copia y pega el siguiente shortcode en cualquier página, entrada o widget donde desees mostrar la interfaz:
   `[nascor_collaboration]`
